2.1. Concepto

Nos piden que hagamos un programa para gestionar gatos de los albergues que hay en Murcia. Para guardar sus nombres sera cansino crear una variable por cada uno. Imaginate que hay 300, sera inviable. Para estas situaciones necesitamos una estructura de datos ms cmoda.

Las variables que creamos ocupan un espacio en la memoria de nuestro ordenador. En cuestin de rendimiento si tenemos que crear 300 variables tenemos que avisar 300 veces que reserve espacio, no es eficiente. Sin embargo hay una forma para decir que nos reserven 300 espacios de una sola vez. Ya no hay 300 avisos, slamente uno con lo que ganamos eficiencia.

Los array o vectores solucionan nuestro problema.
